### Step 8: Common Tasks

#### Report a Deviation
1. Go to **Deviations**
2. Click **"Report Deviation"**
3. Enter details (title, type, priority)
4. Click **"Submit"**
5. View in deviation list

#### Create a Batch
1. Go to **Batch Management**
2. Click **"New Batch"**
3. Enter batch details
4. Click **"Create"**

#### Register a Complaint
1. Go to **Complaints**
2. Click **"Register Complaint"**
3. Fill in complaint details
4. Select severity
5. Click **"Register"**

#### Request a Change
1. Go to **Change Control**
2. Click **"Request Change"**
3. Enter change details
4. Select type and priority
5. Click **"Submit"**

#### View Audit Trail
1. Go to **Audit Trail**
2. Use search box to find records
3. Filter by user, action, date
4. View timestamps and IP addresses
5. Export for compliance

#### Add a User
1. Go to **User Management**
2. Click **"Invite User"**
3. Enter email and select role
4. Click **"Send Invite"**
